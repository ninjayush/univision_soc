# Lab:

### Build a three-stage dashboard mock.

**How to run the dashboard:**
This project uses Docker to avoid polluting the host system with Node.js and NPM. You can start the dashboard using `docker compose`:

```bash
cd week3lab
docker compose up
# If you encounter permission errors on WSL, run: sudo docker compose up
```

The application will hot-reload and become accessible in your browser at [http://localhost:5173](http://localhost:5173).
